Panduan Deployment AISwarm Microfish

Panduan lengkap dan mendalam untuk melakukan deployment proyek AISwarm Microfish. Panduan ini disusun agar Anda dapat menjalankan layanan microservice ini dengan stabil, baik untuk keperluan pengembangan maupun produksi menggunakan PM2.

Proyek ini merupakan layanan berbasis Python (FastAPI/Uvicorn) yang mengintegrasikan kecerdasan buatan (DeepSeek). Ikuti langkah-langkah di bawah ini secara berurutan.

1 Persiapan dan Unduh Source Code

Langkah pertama adalah menyalin repositori dari GitHub ke direktori lokal atau VPS Anda. Pastikan Anda sudah menginstal git.

# Menarik data dari repositori resmi
git clone https://github.com/unobengkel/aiswarm_microfish.git

# Masuk ke direktori proyek
cd aiswarm_microfish

2 Isolasi Lingkungan (Virtual Environment)

Sangat disarankan untuk menggunakan Virtual Environment agar pustaka (library) proyek ini tidak bentrok dengan pustaka sistem global Anda.

# Membuat folder venv sebagai wadah library
python3 -m venv venv

# Mengaktifkan virtual environment
# Jika di Linux/macOS:
source venv/bin/activate

3 Instalasi Dependensi

Setelah lingkungan virtual aktif, instal semua modul Python yang dibutuhkan oleh proyek ini sesuai dengan daftar di requirements.txt.

pip install --upgrade pip
pip install -r requirements.txt

4 Konfigurasi API Key

Layanan ini membutuhkan DeepSeek API Key agar fungsi AI-nya berjalan. Ada dua cara untuk melakukannya:

Cara Langsung (Tidak Disarankan)

Memasukkan key langsung ke dalam file kode. Ini berisiko jika kode Anda terunggah ke publik.

nano main.py   # masuk ke halaman main.py

# Mengedit langsung di dalam file (misal: main.py y)
DEEPSEEK_API_KEY = "sk-your-key-here"

Cara Aman (Disarankan)

Menggunakan environment variable di terminal.

export DEEPSEEK_API_KEY="sk-..."

5 Pengaturan Port dan Host

Untuk memastikan aplikasi dapat diakses dari luar (public IP), Anda perlu menyesuaikan konfigurasi server Uvicorn di dalam file main.py.

  1. Buka file menggunakan editor teks nano: 
    nano main.py
  2. Cari bagian uvicorn.run dan pastikan pengaturannya seperti berikut agar bisa diakses secara global: 
    uvicorn.run(app, host="0.0.0.0", port=8000)
    (Ganti port 8000 jika Anda ingin menggunakan port lain).
  3. Simpan dengan menekan CTRL + O, lalu Enter, dan keluar dengan CTRL + X.

6 Uji Coba Server

Sebelum menjalankan di latar belakang, lakukan pengujian langsung untuk memastikan tidak ada error pada kode atau dependensi.

python main.py
Catatan: Jika muncul pesan "Uvicorn running on http://0.0.0.0:8000", berarti server sudah aktif. Tekan CTRL + C untuk menghentikannya sebelum lanjut ke tahap permanen.

7 Deployment dengan PM2 (Production Mode)

Agar aplikasi tetap berjalan meskipun terminal ditutup atau server melakukan restart, kita akan menggunakan PM2 (Process Manager 2).

  1. Keluar dari Virtual Environment (Opsional, karena kita akan memanggil path-nya secara langsung): 
    deactivate
  2. Jalankan dengan PM2:
    Kita akan memerintahkan PM2 untuk menjalankan main.py menggunakan interpreter Python yang ada di dalam folder venv. 
    pm2 start main.py --name "aiswarm-microfish" --interpreter ./venv/bin/python
  3. Monitor Status:
    Untuk melihat apakah aplikasi berjalan dengan lancar: 
    pm2 list
pm2 logs aiswarm-microfish

Perintah Tambahan PM2:

  • Berhenti: pm2 stop aiswarm-microfish
  • Restart: pm2 restart aiswarm-microfish
  • Hapus dari list: pm2 delete aiswarm-microfish
  • Auto-start saat reboot: pm2 save

Dengan langkah-langkah di atas, AISwarm Microfish Anda sekarang sudah berjalan sebagai layanan latar belakang yang tangguh! Ada bagian spesifik yang ingin kamu ulik lebih dalam, misalnya integrasi ke domain atau setup SSL (HTTPS)?

Catatan Penggunaan & Pemeliharaan

Setelah server berjalan, gunakan panduan ini untuk menjaga kesehatan layanan Anda.

Monitoring Performa

Gunakan tool monitoring internal PM2 untuk melihat penggunaan CPU dan RAM secara real-time.

pm2 monit

Melihat Riwayat Log

Jika ada error atau masalah akses, periksa log aplikasi untuk detail masalahnya.

pm2 logs aiswarm-microfish --lines 100

Tips Penting Setelah Instalasi:

  • Ganti Port Default: Jika port 8000 sudah digunakan aplikasi lain, segera ganti di main.py dan restart PM2.
  • Keamanan API: Jangan pernah membagikan API Key Anda. Jika bocor, segera ganti di DeepSeek dashboard dan update server Anda.
  • Pembaruan Kode: Jika ada update di GitHub, lakukan git pull, lalu pm2 restart aiswarm-microfish.
